Interfaces

Here are some types declared for data send by the server.

Player

{string} name - The player name
{string} team - The player team
{boolean} ready - If the player is currently ready
{number} id - The player id that is assigned to the entity on game-start
{number} player - If it is local player 1 or local player 2

Entity

{number} size - The size of the entity
{x:number,y:number} pos - The position of the entity
{string} direction - One of the following directions: NORTH, SOUTH, EAST, WEST, NONE
{number} id - The entity id
{string} team - The entity team.
{string} name - The entity name.

Tile

Note, the x and y are not coordinates but indexes, to get the tile coordinates multiply index with the cellsize (100)
{string} tile_type - 'none' 'wall' 'stop' What kind of tile it is.
{number} x - The x index of the tile
{number} y - The y index of the tile

Board

{number} players - The max amount of players
{number} width - The board width
{number} height - The board height
{Tile[][]} tiles - The the double tile array, a tile at position x of y can be acces by tiles[x][y]

Game

{Board} board - The game board.
{Map(Entity)} entities - A json object with as key the entity id and as value the entity.
